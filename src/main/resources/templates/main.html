DevNotes — open-source веб-проект, объединяющий несколько ролей:

  • ✦ авторский IT-блог с техническими статьями и заметками о разработке
  • ✦ pet-проект в портфолио, демонстрирующий навыки проектирования и построения приложений
  • ✦ экспериментальная площадка для изучения, тестирования и применения новых технологий

О проекте

Проект реализован на Java 21 с использованием базы данных PostgreSQL.

Основная цель проекта — объединить процесс обучения, разработки и документирования инженерного опыта в одном месте.

Подробности об архитектуре, устройстве приложения и структуре кода доступны на GitHub, где также публикуются обновления и изменения проекта.
